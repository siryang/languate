桥接模式的一个用途

//##Test.h
class TT
{
	private；
		int x, y;
}
###

问题：
	上述代码编译成库文件后，
	将private改为publie就可以正常访问private变量。

解决：
	如果定义一个Struct Self{int x; int y};
	然后再private中使用Self*声明，则能隐藏TT中的私有变量结构。
